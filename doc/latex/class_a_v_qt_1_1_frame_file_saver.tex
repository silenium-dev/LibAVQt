\doxysubsection{Detailed Description}
This class is a simple \mbox{\hyperlink{class_a_v_qt_1_1_i_frame_sink}{IFrame\+Sink}} implementation that saves every n-\/th frame to BMP files. 

\begin{DoxySince}{Since}
0.\+1-\/dev
\end{DoxySince}
While developing custom frame sources or filter, a simple image file output is very useful for debugging. \mbox{\hyperlink{class_a_v_qt_1_1_frame_file_saver}{Frame\+File\+Saver}} provides this feature with custom file prefixes for multiple instances in a single application
