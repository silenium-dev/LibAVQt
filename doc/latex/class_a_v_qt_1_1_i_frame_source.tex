\doxysubsection{Detailed Description}
Base class for every frame sink. 

All subclasses of \mbox{\hyperlink{class_a_v_qt_1_1_i_frame_source}{IFrame\+Source}} represent a possible start of a frame processing pipeline (e.\+g. a decoder, a desktop/window capture, camera).
