Detailed Description

VAAPI accelerated video decoder.

Decodes video from QIODevice into single frames, that are passed to every via registerCallback registered callback. It does not stop when no callbacks are registered, so make sure, that either at least one callback is registered or the decoder is paused, or frames will be dropped
