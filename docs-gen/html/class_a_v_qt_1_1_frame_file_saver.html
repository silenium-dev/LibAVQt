Detailed Description

This class is a simple IFrameSink implementation that saves every n-th frame to BMP files.

Since
0.1-dev

While developing custom frame sources or filter, a simple image file output is very useful for debugging. FrameFileSaver provides this feature with custom file prefixes for multiple instances in a single application

See also
IFrameSink
