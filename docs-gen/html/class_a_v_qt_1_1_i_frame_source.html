Detailed Description

Base class for every frame sink.

All subclasses of IFrameSource represent a possible start of a frame processing pipeline (e.g. a decoder, a desktop/window capture, camera).

Member Enumeration Documentation

◆ CB_TYPE

enum AVQt::IFrameSource::CB_TYPE : int8_t

Used to define accepted callback types of a frame source or filter.

Can be linked with bitwise or to set multiple types

Enumerator
CB_QIMAGE 

Callback type: QImage. When registered with this type, the onFrame(QImage, ...) method will be called.

CB_AVFRAME 

Callback type: AVFrame. When registered with this type, the onFrame(AVFrame, ...) method will be called.

Important: The AVFrame can be every possible software pixel format, so the frame sink/filter has to be able to convert the pixel format if necessary. It has to be freed after usage. If the source processes frames on the GPU, it has to be downloaded to CPU memory before passing to callbacks.
