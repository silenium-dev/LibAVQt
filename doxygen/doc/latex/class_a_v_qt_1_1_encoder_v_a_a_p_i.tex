\doxysubsection{Detailed Description}
VAAPI accelerated video encoder. 

Sink for encoding every type of frame into mpegts file with provided codecs. Same instance cannot be used as callback of both types. {\itshape {\bfseries{Requires constant frame input or else the output will be scrambled. This is preliminary and will be subject to later changes.}}}
